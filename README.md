# Deltaclass - Exam Planning and Scheduling System for University Entrance Exams

Deltaclass is a comprehensive planning and scheduling system for students preparing for Turkish university entrance exams (TYT/AYT). This platform enables students to organize their exam preparations, teachers to track student progress, and counselors to create personalized plans.

## Role-Based User System

The application has three different user roles:

| Role | Description |
|------|-------------|
| **Student** | Creates exam plans, tracks mock exam results |
| **Teacher** | Monitors assigned students' progress, provides analyses |
| **Counselor** | Manages all students, creates individual/group plans |

## Technology Stack

- **Backend**: Laravel 10.x (PHP 8.1+)
- **Frontend**: Bootstrap 5.3, JavaScript, Blade Templates
- **Database**: MySQL/SQLite
- **ORM**: Eloquent
- **Authentication**: Laravel Authentication
- **CSS Framework**: Bootstrap 5
